# Privacy Policy

Effective date: 2026-06-14

3D Game Optimizer is designed for local, privacy-first use.

## Data We Process

| Data Category | Example | Why It Is Needed | Storage | Retention |
|---|---|---|---|---|
| Local settings | selected display profile, quality tier | preserve user preferences | local app storage | until user resets/uninstalls |
| Hardware metadata | display identifiers (EDID signature), GPU vendor | device compatibility recommendations | local cache | refreshed on scan; removable by user |
| Tool execution metadata | tool id, install status, exit code | troubleshooting automation runs | local logs | rolling retention (default 30 days) |

## Data We Do Not Collect by Default

- No account data collection.
- No automatic telemetry uploads.
- No background cloud sync.
- No sale or brokerage of user data.

## Optional Network Operations

When a user explicitly requests data refresh (for example, compatibility seed update checks), network access may occur. Requests should be transparent in the UI and limited to minimum required metadata.

## User Controls

- Reset local settings and cache.
- Delete local logs.
- Disable optional network refresh behavior.

## Security and Minimization

- Keep logs operational and non-personal.
- Avoid collecting filenames, chat content, or full system inventories.
- Store only fields required for compatibility, presets, and automation reliability.

## Contact

For privacy concerns, contact project maintainers through repository security/contact channels.
